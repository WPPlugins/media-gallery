Easy generation of simple image galleries from media categories

== Description ==

Generate image galleries from wordpress media categories

* Add custom taxonomy gallery to media
* Generate image gallery from media gallery taxonomy
* Thickbox popup for single images
* TinyMCE Button to easily include image gallery into post or page

== Installation ==

To install the plugin do the following steps

1. Install and activiate the plugin via the admin plugin page.
